# 🚀 Install TabbyAnnounce

Copy and paste the command for your operating system to get started.

### 🐧 Linux & 🍎 macOS
```bash
git clone https://github.com/Stupidgoat2772/TabbyAnnounce.git && cd TabbyAnnounce && npm run setup
```

### 🪟 Windows (PowerShell)
```powershell
git clone https://github.com/Stupidgoat2772/TabbyAnnounce.git; cd TabbyAnnounce; npm run setup
```

---

## What happens next?
1. **Dependencies** will be installed automatically.
2. A **.env** file will be created for you.
3. Follow the instructions in the [README.md](README.md) to add your Discord token and Google Spreadsheet link!
